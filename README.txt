
# Simple To-Do App

This is a lightweight To-Do list web app (HTML/CSS/JS) that uses localStorage to persist tasks.
Files included:
- index.html
- style.css
- script.js

How to run:
1. Open `index.html` in your browser.
2. Optionally deploy to a static hosting (Vercel, Netlify, GitHub Pages).

Features:
- Add, edit, delete tasks
- Mark tasks as completed
- Clear completed tasks
- Export tasks as JSON
